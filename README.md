## How to Setup
The following are how to setup for macOS in development environment.

1. Install nodenv
2. Install yarn via `npm install -g yarn`
3. Run `yarn`
4. Run `nodenv install` in the project root directory
5. Run `yarn start` to run instant HTTP server
